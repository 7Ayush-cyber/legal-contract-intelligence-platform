# Legal Contract Intelligence Platform

Enterprise-grade Legal AI platform built using:

- Hybrid Search (FAISS + BM25)
- Retrieval Augmented Generation (RAG)
- Legal Clause Extraction
- Risk Detection
- Missing Clause Analysis
- Contract Comparison
- Streamlit

## Features

### Contract Q&A
Ask legal questions across hundreds of contracts.

### Contract Analysis
- Clause Extraction
- Risk Detection
- Missing Clause Detection
- Executive Summary

### Contract Comparison
Compare two contracts and identify key legal differences.

## Tech Stack

- Python
- Streamlit
- LangChain
- OpenAI / Bedrock
- FAISS
- BM25
- CUAD Dataset
